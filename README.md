# Café del Code

Proyecto: entornos/cafe-del-code

Descripción
- Repositorio con ejercicios y configuraciones para el módulo de entornos de desarrollo.
- Objetivo: centralizar código, scripts y apuntes relacionados al curso.

Requisitos
- Git
- Node.js (opcional, según ejercicios)
- Python (opcional, según ejercicios)

Instalación
1. Clonar el repositorio:
    git clone <url-del-repositorio>
2. Entrar al directorio:
    cd entornos/cafe-del-code

Estructura típica
- README.md — este fichero
- /ejercicios — ejercicios y prácticas
- /scripts — scripts útiles (instalación, limpieza)
- /docs — documentación y apuntes

Uso
- Revisar el README de cada subcarpeta para instrucciones específicas.
- Ejecutar scripts desde la raíz con:
  ./scripts/nombre-script.sh
  o
  node scripts/nombre-script.js

Contribuir
- Crear una rama por feature: git checkout -b feat/nombre
- Hacer commits claros y descriptivos
- Enviar pull request explicando cambios
